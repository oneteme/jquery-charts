Principe

Une seule API, plusieurs moteurs

Le même composant Angular, les mêmes field() et rangeFields() — seul l'import change.

ECharts 
import { ChartComponent }
  from '@oneteme/jquery-echarts';

<chart
  type="bar"
  [config]="config"
  [data]="data"
></chart>
