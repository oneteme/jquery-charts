Slice panel — usage avec un graphique

Le composant <slice-panel> est découplé de <jquery-table>. Il peut s'utiliser seul pour filtrer n'importe quel jeu de données : graphique, liste, carte, etc.

API rapide

Inputs

PropriétéTypeDéfautDescription
sliceConfigsSliceConfig<T>[][]Groupes de filtres. Chaque entrée génère un panneau de catégories.
dataT[][]Données sources. Requis lorsque columnKey est utilisé pour l'inférence automatique.
showCountsbooleantrueAffiche le compteur de lignes sur chaque catégorie. Désactiver pour les graphiques.
alwaysShowbooleanfalseAffiche le panneau même si data est vide (utile quand la data transite ailleurs).
showTogglebooleantrueAffiche le bouton de repli global du panneau.
collapsedByDefaultbooleanfalseReplie le panneau à l'initialisation.

Outputs

ÉvénementTypeDescription
filterChange(row: T) => booleanPrédicat combiné de tous les slices actifs. À passer directement à Array.filter().
activeKeysChangestring[][]Clés actives par slice (index 0 = slice 0). Alternatif à filterChange pour alimenter les paramètres d'une requête graphique.
collapsedChangebooleanÉmet true quand le panneau est replié, false quand il est ouvert.

SliceConfig<T>

PropriétéTypeDescription
titlestringTitre affiché en en-tête du groupe.
columnKeykeyof TClé de propriété. Les catégories sont générées automatiquement depuis les valeurs distinctes.
categoriesSliceCategory<T>[]Catégories manuelles. Mutuellement exclusif avec columnKey.
multiSelectbooleanSélection multiple dans ce groupe (défaut : false).

Démo interactive

Cliquez sur un filtre pour voir le graphique se mettre à jour en temps réel.
